# Video assets

Drop real video files here. The components reference these paths:

- `hero-placeholder.mp4` — Hero section intro video (default in `VideoPlayer`)
- transformation / testimonial / Q&A videos — pass a `src` prop to `<VideoPlayer />`

Until real files exist, `<VideoPlayer />` shows a styled poster + play button.
Add matching poster images and pass them via the `poster` prop for best UX.
